📋 Cara Order untuk Pelanggan
1
Scan QR Code
Buka kamera HP → scan QR Code Berkah Tani → otomatis buka katalog
2
Pilih Produk
Browse katalog, filter kategori, cari produk → klik "+ Keranjang"
3
Isi Data & Checkout
Isi nama, no. HP, alamat → pilih metode bayar → klik "Konfirmasi Pesanan"
4
Konfirmasi via WhatsApp
Klik tombol WA → pesan otomatis terkirim ke toko → tunggu konfirmasi admin
✓
Pesanan Diproses Admin
Admin lihat di menu "Pesanan Online" → konfirmasi → proses → selesai
